# Clever Task

You need ```http-server``` to be installed:
```sh
$ sudo npm install -g http-server
```
### Installation
```sh
$ git clone https://github.com/ocnskyy/cleverTask.git
$ cd cleverTask
$ npm install
```
The app is available at <http://localhost:9999/#/main>
### Development
```sh
$ gulp watch
```
