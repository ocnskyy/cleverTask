# Clever Task
### Requirements
You need `http-server`, `bower` and `gulp`to be installed:
` $ sudo npm install -g http-server gulp bower

### Installation
```sh
$ git clone https://github.com/ocnskyy/cleverTask.git
$ cd cleverTask
$ npm install
```

The app is available at <http://localhost:9999/#/main>
### Development
```sh
$ gulp watch
```
